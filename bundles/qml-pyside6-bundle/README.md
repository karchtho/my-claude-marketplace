# qml-pyside6-bundle

QML + PySide6 development expertise for Thomas's widget-based desktop dashboard.

## Skills

### `thomas-qml-dashboard`

Activates when working on the dashboard project — QML components, theme system, sidebar,
tab management, widget architecture, Python bridge patterns.

**Triggers on:** QML, WidgetShell, WidgetContent, WorkspaceStore, Theme singleton,
BaseWidget, sidebar, Flickable, snapped/floating mode, design tokens, qmldir,
PySide6 @Slot/@Property, QAbstractListModel, and more.

**Reference files** (loaded on demand):
- `references/architecture.md` — Widget architecture, WidgetShell/WidgetContent interface, tab system
- `references/theme-system.md` — Full token system, four themes, persistence, dark-flash fix
- `references/python-bridge.md` — PySide6 @Slot, @Property, @QmlElement, signal/slot wiring

## Installation

Add to your project's `.claude/plugins.json` or reference from the marketplace registry.
